# FL-based Intrusion Detection System for Vehicle IoT

## Phase-1: Binary IDS (Completed)

This project implements a Federated Learning based Intrusion Detection System (IDS) for Vehicle IoT environments using the CIC-IDS2018 dataset.

### Phase-1 Highlights
- Binary classification: Benign vs Attack
- PCC-based feature selection
- Centralized Logistic Regression baseline
- Federated Learning with FedAvg (5 clients)
- Significant accuracy improvement using FL

> Large datasets are excluded from this repository.

## Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
- Federated Learning (FedAvg)
