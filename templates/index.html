// GAUSSIAN MECHANISM — WEIGHT CLIPPING + NOISE INJECTION
Parameters
Algorithm
INPUT: weights W, clip C, noise σ
STEP 1 — Sensitivity Bounding:
norm = ||W||₂
if norm > C: W = W × (C / norm)
STEP 2 — Noise Injection:
η ~ N(0, σ²C²)
W_dp = W_clipped + η
OUTPUT: W_dp → sent to server
